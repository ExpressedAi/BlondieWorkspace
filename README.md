# Blondie Workspace

Deployed on Vercel with Next.js (App Router).

## Run

```bash
npm run dev
```

## Deploy
- Connect GitHub repo in Vercel
- Push to main -> auto deploy
- Preview deploys for PRs

## Env
```bash
# link local to Vercel project
vercel link
# pull envs
vercel env pull .env.local
```
